Controls (Gamepad/Keyboard)

  • Move: Left Stick / WASD Keys
  • Camera: Right Stick / Mouse
  • Attack: Right Shoulder / Left Mouse Button
  • Block: Left Shoulder / Right Mouse Button
  • Lantern: West Button / Q Key
  • Interact: South Button / E Key
  • Use Potion: North Button / F Key
  • Pause: Start Button / Escape Key (To Be Changed)

My Contributions

I made this game almost entirely on my own for a second year school project. While I did not create any of the asssets, I do have a list of the assets used on the Itch.io page. Aside from that, I did almost everything else myself. This includes all of the level design, and almost all of the programming. Another student contributed a bit to programming the game sounds with my help, but everything else was done by myself in under 2 months.
